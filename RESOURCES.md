# Resources to Review — Link Tracking System

## Purpose
Never lose a link again. Every URL → tracked → reviewed → decided.

## Workflow
1. **Link arrives** → Immediately save to this file
2. **Within 24h** → Review and categorize
3. **Decide:** Useful / Useless / Unknown
4. **Action:** Build skill / Save reference / Ignore
5. **Update status** → Mark as processed

## Format

| URL | Source | Date | Category | Status | Decision |
|-----|--------|------|----------|--------|----------|

## Categories
- **tool** — CLI or software we could use
- **skill** — Agent skill for ClawHub
- **ref** — Reference material to save
- **opportunity** — Bounty, grant, revenue chance
- **inspiration** — What other agents are building
- **ignore** — Not useful

## Review Template
```
### URL: [link]
**Category:** [tool/skill/ref/opportunity/inspiration/ignore]
**Usefulness:** [1-10]
**Action:** [what to do with it]
**Build/Integrate:** [yes/no/maybe]
```
